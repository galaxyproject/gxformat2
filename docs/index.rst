Galaxy Workflow Format 2
========================

**gxformat2** is the reference library for Galaxy's human-friendly YAML workflow format.
It converts between Format2 (YAML) and native Galaxy (.ga) representations, provides
linting, normalization, and Pydantic-based validation.
